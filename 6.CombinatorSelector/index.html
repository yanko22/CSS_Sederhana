Combinator Selector

  • Combinator adalah sesuatu yang menjelaskan relasi antar selector
  • Terdapat 4 jenis combinator dalam CSS:
    1. Descendant Combinator (spasi)
    2. Child Combinator (>)
    3. Adjacent Sibling Combinator (+)
    4. General Sibling Combinator (~)

Descendant Combinator Selector

  • Descendent Selector adalah selector untuk memilih element anak dari element yang dipilih
  • link referensi

pada contoh gambar dibawah ini menunjukan kita memilih element anak dari element yang dipilih

Contoh Pengguaan Descendant Combinator Selector

            
                HTML :
                    <div>
                        <article>
                            <p> ini paragraf </p>
                         </article>
                        <p> ini paragraf </p>
                    </div>
                    <p>ini paragraf</p>

                CSS :
                    div p{
                        color: red;
                    }

dari hsail di atas kita dapat melihat dimana semua paragraf yang berada di dalam element div akan dipilih meskipun p pertama berada pada elemen article

pada contoh dibawah ini artinya kita memilih element p yang menjadi anak langsung dari element div

Adjacent Sibling Selector

  • Adjacent Sibling Selector digunakan untuk memilih element setelah element yang dipilih
  • Sibling (sudara) element harus memiliki perent elemnt yang sama
  • Link referensi

pada gambar dibawah ini artinya memilih element p yang diletakkan setelah element div dimana p dan div memiliki parent yang sama

Contoh Code :

            
                HTML :
                    <div>
                        <article>
                            <p> ini Tidak Dipilih </p>
                         </article>
                        <p> ini Tidak Dipilih </p>
                    </div>
                    <p>ini dipilih</p>
                    <p>ini Tidak dipilih</p>

                CSS :
                    div + p{
                        color: blue;
                    }
