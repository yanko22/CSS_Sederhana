Combinator Selector

  • Combinator adalah sesuatu yang menjelaskan relasi antar selector
  • Terdapat 4 jenis combinator dalam CSS:
    1. Descendant Combinator (spasi)
    2. Child Combinator (>)
    3. Adjacent Sibling Combinator (+)
    4. General Sibling Combinator (~)

Descendant Combinator Selector

  • Descendent Selector adalah selector untuk memilih element anak dari element yang dipilih
  • link referensi

pada contoh gambar dibawah ini menunjukan kita memilih element anak dari element yang dipilih

Contoh Pengguaan Descendant Combinator Selector

            
                HTML :
                    <div>
                        <article>
                            <p> ini paragraf </p>
                         </article>
                        <p> ini paragraf </p>
                    </div>
                    <p>ini paragraf</p>

                CSS :
                    div p{
                        color: red;
                    }

Child Combinator Selector

  • Child Selector adalah selector untuk memilih child / anak dari element yang dipilih
  • link referensi
pada contoh dibawah ini artinya kita memilih element p yang menjadi anak langsung dari element div
