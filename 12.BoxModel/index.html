Box Model

  • Saat menggunakan CSS, terdapat konsep yang disebut Box Model

  • Box Model biasanya digunakan untuk mengatur tata letak atau layout pada sebuah elemen

  • Setiap element di HTML, memiliki Box yang terdiri dari content, padding, border dan margin

hal tersebut dapat dipahami melalui gambar dibawah ini

penjelasan dari bagian2 Box Model:

  • Content: bagian yang berisi konten dari sebuah elemen

  • Padding: bagian transparan antara content dan border

  • Border adalah kotak yang mengelilingi content dan padding

  • Margin: bagian transparan di luar border digunakan untuk mengatur jarak antara elemen

Size

  • Saat kita membuat element HTML, kita tahu bahwa beberapa element ditampilkan dalam display block dan beberapa lainnya dalam display inline
  • Dengan menggunakan CSS, kita dapat mengatur height dan width dari sebuah element inline
  • Hal ini penting dikarenakan display dengan bentuk inline tidak dapat diatur ukurannya dengan height dikarenakan besar dari hight akan mengikuti tinggi dari isi kontennya
