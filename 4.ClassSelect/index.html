Belajar Class Select

Class

  1. Kadang ada kasus ingin Menambahkan style CSS ke beberapa element tag yang jenisnya berbeda;
  2. Jika kita harus buat aturannya per tag, maka akan menyulitkan ketika banyak sekali;;
  3. Di HTML, semua tag bisa memiliki atribut class, dan di CSS, kita bisa Menambahkan aturan ke class dengan menggunakan awalan .(titik);

Contoh

    
        HTML :
        <p class="class"> INI PARAGRAF </p>
        
        CSS :
        .class{
                color: red;
                font-size: larger;
        }

INI PARAGRAF

Multiple Class

  1. Nilai dari atribut class bisa menggunakan beberapa nilai, caranya bisa tembahkan pemisah menggunakan spasi

Contoh :

        
            HTML :
                <h1 class="red upper"> INI PARAGRAF </h1>
                <h2 class="red upper"> INI PARAGRAF </h2>

            CSS :
                .red{
                    color: red;
                }
                .upper{
                    text-transform: uppercase;
                }
