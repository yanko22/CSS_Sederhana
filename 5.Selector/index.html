Selector

  • Cara untuk memilih elemen untuk ditambahkan style yaitu menggunakan tag # untuk ID dan . untuk class;
  • Memilih elemen pada CSS disebut sebagai selector;
  • Ada banyak jenis selector dalam CSS;

Simple Selectors

  • simple selector adalah selector untuk memilih elemen berdasarkan nama (tag), #id atau .class;
  • Jika kita ingin membuat selector unutk beberapa element, kita bisa gunakan , (koma) sebagai pemisah;
